Chine Entrée pour les citoyens chinois

Les citoyens chinois n'ont pas besoin de visa pour entrer en Chine continentale. A valable Passeport chinois est suffisant pour entrer dans tout port international. Les résidents chinois à l'étranger peuvent également utiliser leur passeport pour voyager de retour.

Détails d'entrée pour les citoyens chinois

  • Pas de visa requis — le passeport chinois accorde une entrée sans restriction
  • Porter un passeport chinois valide (non expiré)
  • Pour les personnes ayant une double nationalité: la loi chinoise ne reconnaît pas la double nationalité; entrez sur votre passeport chinois

Comment faire une demande

  1. Assurez-vous que votre passeport chinois est valide pour la durée de votre séjour.
  2. Passez directement par la voie des passeports chinois / citoyens chinois à l'immigration.
  3. Présentez votre passeport chinois pour l'estampillage.

Documents requis

  • Passeport chinois valide
